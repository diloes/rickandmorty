## Rick y Morty

La webapp consiste en consumir la API de rick y morty. 

He utilizado React:
 - useState
 - useEffect
 - react-router-dom

Para los estilos(muy mejorables, lo sé... xD) he utilizado css modules para que no haya
conflictos con los nombre de clases.
